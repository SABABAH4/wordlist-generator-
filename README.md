#  Wordlist Generator (Python)

A simple and customizable Python script to generate wordlists.

## Features
- Custom character sets (letters, numbers, symbols)
- Control word length
- Multiple output files
- Easy configuration

##  Configuration
Edit the variables inside `generator.py`:

- WORD_LENGTH
- WORDS_PER_FILE
- NUMBER_OF_FILES

##  Usage

```bash
python generator.py
